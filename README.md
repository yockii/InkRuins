# 墨墟 InkRuins
一款基于雪花写作法及AI技术的网络文学创作平台，旨在利用雪花写作法的优势，结合AI技术的强大能力，为用户提供创作网络文学的便捷工具。

本平台适用于希望编写完整的网络文学作品的用户，但不适用于写无限流的网络文学。

## 使用方法
1. 注册账号并登录平台。
2. 创建新的网络文学项目，填写项目名称、简介、项目类型等基本信息。
3. 进入项目界面，按照雪花写作法，先完成一句话的作品主要内容。
4. 扩写，可使用AI或自行扩写成5句话以内的一段摘要（基本设定1句、主要灾祸3句、故事结局1句）。
5. 定义角色，包括角色的名称、类型、性格、动机、目标等等，可以由AI根据摘要生成后进行修改。
6. 定义故事中人物所需要的可变属性，可由AI生成并根据需要进行修改。——这里可能对人物关系、地理信息、道具等都可以做设定。
7. 将4中的摘要每一句话扩写为一小段，可利用AI生成后修改。
8. 将角色的人物宝典进行扩写，将其背景、经历、性格、外貌等进行扩展，让人物更立体、更有深度，可用AI生成后修改。
9. 将6中的每一小段进行扩展，可利用AI生成后修改。
10. 将8中的每一段故事概要，形成场景清单（故事节点），并就该故事节点的属性变化进行记录，可由AI生成。
11. 根据故事线开始动笔实际写作，可由AI辅助或生成

注：如果需要使用某些写作手法如倒叙等，则需要自行写作，AI可以生成但可能需要明确的提示。故事线（故事节点）应该是按时间顺序展开的，但实际写作可以从故事节点中的某一段开始发展，而之前的片段则作为背景或铺垫，甚至作为下一部小说的故事？

注2：在创作过程中，可以随时根据需要调整各类数据
